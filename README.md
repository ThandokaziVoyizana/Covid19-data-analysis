# Covid19-data-analysis
Exploratory data analysis of global COVID-19 rates using Python, Pandas, and Matplotlib.

Project Overview
This project analyzes global COVID-19 infection and mortality trends using Python data analysis tools. The notebook explores patterns, trends, and insights from COVID-19 datasets through data cleaning, visualization, and exploratory analysis.
Tools & Libraries
Python
Pandas
NumPy
Matplotlib
Jupyter Notebook
Features
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
COVID-19 trend visualizations
Statistical summaries
Country-level comparisons
